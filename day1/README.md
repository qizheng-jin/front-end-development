FTP和HTTP协议的区别：
目的不同：
ftp 文件传输协议 远程两台计算机如何传输文件
http 超文本的传输协议， 网页，从别人的服务器抓取内容来展现
https 安全的http，加密base64,网页加载速度慢, 可以预防DDOS攻击.
